# C+ Language Syntax

C+ uses a modern, clean syntax designed for readability. Every statement in C+ must end with a semicolon (`;`).

## Variables

Variables are declared using the `let` keyword:

```cplus
let name = "Developer";
let version = 1.0;
```

## Functions

Functions are defined using the `def` keyword followed by the function name and parameters in parentheses. The body of the function is enclosed in curly braces `{}`.

```cplus
def add(a, b) {
    return a + b;
};
```

## Data Types

C+ supports several fundamental data types:
- **Numbers**: `let x = 10;`
- **Strings**: `let s = "Hello";`
- **Booleans**: `True`, `False`
- **None**: Represents the absence of a value.
- **Arrays**: `let list = [1, 2, 3];`
